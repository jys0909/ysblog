### 1、两位小数以内的正实数
```
^(([0-9]+)|([0-9]+\.[0-9]{1,2}))$
```

### 2、中文、字母、数字、或者下划线
```
/^[\u4e00-\u9fa5A-Za-z0-9_]+$/
```

### 3、电话号码
```
/^(0[0-9]{2,3}\-)?([2-9][0-9]{6,7})+(\-[0-9]{1,4})?$|(^(13[0-9]|15[0|3|6|7|8|9]|18[8|9])\d{8}$|(^400-([0-9]){1}([0-9-]{6})([0-9]){1})$)/

```

### 4、网址
```
/^(f|ht){1}(tp|tps):\/\/([\w-]+\.)+[\w-]+(\/[\w- ./?%&=]*)?/
```

### 5、邮箱
```
\w[-\w.+]*@([A-Za-z0-9][-A-Za-z0-9]+\.)+[A-Za-z]{2,14}
```
### 6、双字节字符
```
[^\x00-\xff]
```

### 7、时分秒
```
([01]?\d|2[0-3]):[0-5]?\d:[0-5]?\d
```
